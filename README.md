# Style_Studio

An innovative e-commerce platform, built entirely by our team using the MERN stack that enhances the online shopping experience by integrating a virtual try-on feature.  
 

## Project Overview

Project Type: Full-Stack Web Application using MERN
Domain: E-Commerce + Virtual Try-On

## Tech Stack:
Frontend: React.js, HTML, CSS
Backend: Node.js, Express.js
Database: MongoDB
Virtual try-on feature: Three.js

 
 

## Key Features

User Authentication - Secure login and session handling<br>
Home Page - Clean and aesthetic landing page<br>
Shop Module - Browse clothing items<br>
View product details<br>
3D Virtual Try-On - Visualize how clothes fit on a virtual avatar<br>
Cart & Checkout - Add/remove products<br>
Secure Payment Integration - Safe and reliable payment process<br>
Informational Pages<br>
About Us<br>
FAQ<br>
Contact Page<br>

 
 
## How to Run the Project?
### Clone the repository
git clone https://github.com/SanjanaS10/Style_Studio.git

### Navigate to the project folder
cd Style_Studio

### Install dependencies
npm install

### Start the development server
npm start
